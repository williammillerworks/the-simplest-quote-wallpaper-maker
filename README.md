# the-simplest-quote-wallpaper-maker

25-11-25: [Claude prototype](https://claude.ai/public/artifacts/59f29b20-8b9e-464e-8c7b-87ed9c132737)

## 1. Project Overview
A simple and aesthetic app for generating personalized iPhone wallpapers. Users can select meaningful quotes and combine them with customizable backgrounds to create and save beautiful wallpapers, similar to Quiet Quote.

**Target Audience**: iPhone users who appreciate minimalist, inspirational, and emotional content

---

## 2. Core Features

### 2.1 Quote Selection
- Curated quote library organized by categories (inspiration, mindfulness, growth, daily life, etc.)
- Custom quote input functionality
- Quote search and filtering options

### 2.2 Background Customization
- **Background Colors**: Solid colors or gradient options
- **Text Styling**: Font selection, size, color, and alignment adjustments
- **Layout Options**: Text positioning (top, center, bottom, custom)
- **Decorative Elements**: Optional minimal decorations (lines, dots, simple icons)

### 2.3 Real-Time Preview
- Live preview in iPhone screen dimensions (1170x2532px)
- Dark/Light mode preview toggle

### 2.4 Save and Share
- Save wallpaper to device gallery
- Share to social media (Instagram, Twitter, etc.)

---

## 3. Design Direction

**Aesthetic**: Minimalist and calming
- Primary backgrounds: white, off-white, beige
- Accent colors: charcoal, deep gray, or soft pastels
- Typography-focused design

**UI/UX**: Intuitive and clutter-free
- Clear call-to-action buttons
- Simple controls (sliders, color pickers, toggles)
- Smooth interactions and transitions

---

## 4. Technology Stack

- **Frontend**: React or React Native (web or mobile)
- **Drawing/Canvas**: HTML5 Canvas or Three.js
- **Storage**: Local storage or cloud backend
- **Device Compatibility**: Standard iPhone resolution (1170x2532px)

---

## 5. User Flow

1. **App Entry** → Home screen with options
2. **Select Quote** → Browse curated library or enter custom text
3. **Customize Background** → Adjust colors, fonts, positioning
4. **Preview** → Real-time visualization
5. **Save/Share** → Download to gallery or share on social media

---
